# RockyBot

## Multipurpose Discord bot hosted on <https://replit.com/>

### Installing Dependencies

Install poetry through pip:

```sh
pip install poetry
```

Then simply execute:

```sh
poetry install
```

to install dependencies from the poetry.lock file
